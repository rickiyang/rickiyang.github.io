Scala文件读写（五）

发表于 2018-04-29 | 分类于 scala | 次
| 字数统计: 452 | 阅读时长 ≈ 1

1.读取文件

在Scala中有两种基本方式可以打开并读取文件：

  1. 使用一个仅一行的代码，这种方式有个副作用-文件不会被自动关闭，对于那些生命周期很短的程序比如shell脚本可以使用。
  2. 使用稍长些的代码，确保文件流会被关闭。

使用简洁语法：

var fileName = "c:/1.txt"
//每次读取一行
for(line <- Source.fromFile(fileName).getLines()){
  println(line)
}
//将所有的行转为一个列表
val txt = Source.fromFile(fileName).getLines().toList
println(txt)
//将所有的内容当做一个字符串读取
val txt1 = Source.fromFile(fileName).getLines().mkString
println(txt1)

结果：

白日依山尽 尽
黄河
List(白日依山尽 尽, 黄河)
白日依山尽 尽黄河

Process finished with exit code 0

文件流会伴随JVM的生命周期。

确保文件流会被关闭的步骤：

var fileName = "c:/1.txt"
val bufferSource = Source.fromFile(fileName)
for(line <- bufferSource.getLines()){
  println(line)
}
bufferSource.close()

需要手动的调用文件流的close方法。

2.写入文件

Scala不提供写入文件的能力，但是你可以使用Java的字符流来写文件，比如：BufferedWriter,PrintWriter,或者FileWriter。

var fileName = "c:/1.txt"
val pw = new PrintWriter(new File(fileName))
pw.write("我是PrintWriter")
pw.close()

val fileWriter = new FileWriter(fileName)
fileWriter.write("我是FileWriter")
fileWriter.close()

val bufferedWriter = new BufferedWriter(new FileWriter(fileName))
bufferedWriter.write("我是BufferWriter")
bufferedWriter.close()

val outputStreamWriter = new OutputStreamWriter(new FileOutputStream(fileName))
outputStreamWriter.write("我是outputStream")
outputStreamWriter.close()

3. 序列化

Scala的序列号方式和Java一样，继承Serilizable接口。但是在实现上有一些区别：

@SerialVersionUID(100L)
class User extends Serializable{

}

需要将@SerialVersionUID(100L)标注添加到类上。因为Serializable是一个特质，所以可以将他混入任何类中，即使这个类已经继承了别的类：

@SerialVersionUID(100L)
class User extends OldUser with Serializable{

}
